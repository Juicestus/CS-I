CS-1-Midterm
============
A collection of code to review tricky things that could be on the exam.

Substrings.java   -> Examples for String.substring
CompareTO.java    -> Examples for String.compareTo
MinMax.java       -> Examples for Math.min and Math.max
Formatting.java   -> Examples for C style string formatting (printf)
Conditionals.java -> Examples for if, else blocks
